# 📚 Tutorly — Mobile Tutor Booking App UI

Tutorly is a modern and aesthetic mobile app UI designed using Figma.  
This project focuses on creating a clean and user-friendly tutor booking experience with soft pastel colors, modern layouts, and smooth UI flow.

---

# ✨ Features

- Clean and modern UI
- Aesthetic pastel design
- Tutor discovery experience
- Tutor details screen
- Session booking flow
- Booking confirmation screen
- User profile screen
- Mobile-first design approach

---

# 📱 Screens Included

1. Welcome Screen  
2. Home Screen  
3. Tutor Details Screen  
4. Booking Screen  
5. Confirmation Screen  
6. Profile Screen  

---

# 🎨 Design Style

- Minimal UI
- Soft lavender theme
- Rounded cards & buttons
- Clean typography
- Smooth spacing and alignment
- Modern mobile app aesthetics

---

# 🛠️ Tools Used

- Figma
- Unsplash Plugin
- Feather Icons

---

# 🎯 Project Goal

The goal of this project was to practice:
- UI/UX Design
- Mobile App Design
- User Flow Design
- Visual Hierarchy
- Modern Design Systems

---

# 💜 About The Project

Tutorly was designed as a student UI/UX practice project to create a premium-looking tutor booking application interface with a simple and engaging user experience.
